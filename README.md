# WRU

## 概要

大学生でも使える電子名刺の作成・交換ができるスマホアプリケーション。  
アプリ内で自分の名刺を作成する。QRコードを介して、自分の名刺を渡すことと、相手の名刺を受け取ることができる。  

## 実装機能

- サインアップ/サインイン
- ログイン後状態維持
- QRコード生成/読み取り
- 名刺作成
- 名刺受け取り時メモ
